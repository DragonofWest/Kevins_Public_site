What are HTML Entities

From what I have gathered, Enitities are pieces of code that are there to help you write things things to the page that can be mistaken for code by the browser. For instance we use "Quatations" in our code all the time. Enitities are a code to insert these "items" into the page with out getting them mixed up with the code. There are generally two ways of putting them in the site. By calling their # or by their name. All entities have a #, but not all of them have a call name.
