What are HTML Entities

From what I have gathered, Entities are pieces of code that are there to help you write things things to the page that can be mistaken for code by the browser. For instance we use "Quatations" in our code all the time. Entities are a code to insert these "items" into the page without getting them mixed up with the code. There are generally two ways of putting them in the site. By calling their # or by their name. All entities have a #, but not all of them have a call name.

You may be wondering how to insert these Entities into your work. Well they each come with a call number. To start this call number we begin with & symbol, followed by either the name or the # and number. Then we end with a ;. These will pop up in your text as the symbol you are looking for.
